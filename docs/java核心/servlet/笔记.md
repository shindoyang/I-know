http协议的特点

![火狐截图_2020-10-02T12-35-18.809Z](笔记.assets/火狐截图_2020-10-02T12-35-18.809Z.png)

1、简单快速：定义了数据规范，键值对

2、灵活：content-type ： json  或者  text

3、无连接，http1.1  之前是短链接

4、无状态：不会保存上一次访问的状态

5、支持B/S 和 C/S 架构 ： 浏览器或者客户端都是同样的http协议
