* Propagation 传播特性
* Isolation 隔离型-和数据库的隔离性一致
* readOnly 只读，在多次读的事务中，如果要防止ABA的问题，有使用场景
* rollbackFor
* rollbackForClassName
* notRollbackFor
* notRollbackForClassName
