其实是临时镜像，一层层叠加功能，最后的一层就是完整的镜像。

临时镜像只能用于镜像的构建，不可以直接使用。

有点像RGB游戏里面的存档点



Docker在使用Dockerfile构建镜像的过程中采用了两种机制：分层和系统快照

1. 分层：按层进行堆叠，最后全部完成才是完整的容器
2. 快照，每一步采用临时镜像存档（优点：临时容器在构建过程中是可以重用的）
